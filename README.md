# 好咖 Cafe
以咖啡電商為主題的 SPA 網站，使用Vue框架練習前端開發

## 前台
商品陳列、分類陳列、商品搜尋、單一商品介紹
購物車（新增項目、編輯數量、刪除項目、套用優惠券、運費計算）
收藏功能
訂單查詢

## 後台
管理者登入 / 登出、驗證登入狀態
產品管理
訂單管理
優惠券管理

### 使用技術
以 Vue 3 與 Vue Cli 架設開發。
前台手刻 CSS，後台使用 Bootstrap 5 框架，以進行切版。
加入 ESLint standard 統一程式碼結構。
加入 Pinia 傳遞跨元件資料，作為狀態管理工具。

### 使用套件
Vue 3
Vue Cli
Vue Router
Vue Axios
vue-loading-overlay
Vee Validation
pinia
Bootstrap 5
Font Awesome
Animate.css
